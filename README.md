# Projetos DataCamp

**Nota:** Este repositório reúne os projetos aplicados desenvolvidos ao longo das trilhas de certificação que concluí no [DataCamp](https://www.datacamp.com/) — todas com conteúdo técnico integralmente em inglês. Cada pasta contém um projeto independente, com seu próprio notebook/script e, quando aplicável, os dados utilizados.

## Sobre

Estes projetos foram desenvolvidos como parte das certificações DataCamp que concluí (Fundamentos de Programação Python, Programação Python, Desenvolvedor Associado Python, Desenvolvedor Python, Associate AI Engineer, e Cientista de Dados Associado em Python), cobrindo análise e visualização de dados, automação, integração com APIs de IA, e boas práticas de programação em Python.
